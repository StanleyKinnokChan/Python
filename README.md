## About Preppin’ Data ✨📊
Preppin’ Data is a weekly data preparation challenge, which aims at improving the ability of analyst to transform the data into the useful format for downstreaming BI tool consumption and data visualization.  It originally targets people who use Tableau prep. As the data transformation logic still applies no matter which tool you use, here I demonstrate my skill of using **Python** to transform the data. 

The techniques are not limited to:
- File I/O (multiple files/ multiple tabs)
- String manipulation (regex & standard)
- Calculation
- Conditional filtering
- Multi-row manipulation
- Datetime handling
- Aggregation
- Join & Union
- Pivot & unpivot
- Duplication Elimination
- Looping and Iteration
- Scaffolding

**Pandas**, **Numpy** and **Datetime** are three main libraries that I used.
